# AI-Powered Language Translation Tool

This project is an AI-powered language translation application developed using Python, Gradio, and Deep Translator.

## Features
- Translate text between multiple languages
- User-friendly Gradio interface
- Supports English, Tamil, Hindi, Telugu, Kannada, Malayalam, French, Spanish, German, Italian, Japanese, Korean, Chinese, Arabic, and Russian

## Technologies Used
- Python
- Gradio
- Deep Translator
